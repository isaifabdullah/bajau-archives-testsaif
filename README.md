#  Bajau Archives: Digital Heritage Preservation

**Universiti Malaya | WIX2001 THINKING AND COMMUNICATION SKILLS**

A secure digital archive designed for the preservation and study of Sama-Bajau maritime culture. This project bridges indigenous oral traditions with modern digital archiving techniques.

##  Key Features
- **Secure Repository:** Access-key protected archive of traditional songs.
- **Digital Broadsheet:** A high-end reading experience for community stories and oral histories.
- **Local Storage:** Browser-based persistence for offline access and data sovereignty.

##  Technical Stack
- **Frontend:** React + TypeScript + Tailwind CSS + Lucide Icons
- **Data Storage:** cloudStorage (Firebase)
- **Deployment:** Vercel
